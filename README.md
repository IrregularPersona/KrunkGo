# KrunkGo
> Port of the krunker-rs project

## Why

I needed it for a personal project that uses Golang, and I couldn't really use
Rust nor C++ for said project.

## Installation
```bash
go get [github.com/IrregularPersona/KrunkGo]
```

### Notes

Same like my other Krunker API wrappers, you will need a developer API key from
FRVR. You can **kindly** request one from Glen.
